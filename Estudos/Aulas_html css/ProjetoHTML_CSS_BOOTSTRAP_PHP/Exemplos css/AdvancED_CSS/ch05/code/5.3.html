Experience

Principal Cello, Farallones Symphony Orchestra

September 2007–Present

Principal cellist for the Farallones Symphony Orchestra, a small rocky islet off the coast of San Francisco, which is a very lonely spot considering it's just me and the seagulls out here. On the upside, lots of solo opportunities! Building a baroque cello based on 17th Century Cremonese style out of found driftwood and old planks from an abandoned lifeboat.

2003–present

Section cellist for the Rock City Chamber Orchestra, with performances from Western Mt. Diablo to as far east as Lodi. Performances of major composers including Grieg, Dvořák, Beethoven, Bach, and others.
