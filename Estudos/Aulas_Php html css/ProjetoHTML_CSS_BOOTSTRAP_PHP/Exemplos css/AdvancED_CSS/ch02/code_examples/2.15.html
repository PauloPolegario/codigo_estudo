Getting to know screen CSS units

These styles apply to screen and print media:

  • color:green
  • color:darkolivegreen
  • color:85,107,47
  • color:(60%,72%,36%)
  • color:#556B2F
  • font-size:2em
  • font-size:3ex
  • font-size:120%

Pixel units typically only apply to screen media:

  • font-size:36px;

These styles are most effective when used in print media:

  • font-size:0.5in
  • font-size:1cm
  • font-size:10mm
  • font-size:24pt
  • font-size:2pc
