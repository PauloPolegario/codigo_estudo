Count every measure

Some orchestral musicians have more to do than others. Take for instance the difference between the string players and the percussion sections. While the strings almost always have to play frequently in every movement of a given work, the percussion sections often have to sit back and count measures. Often they won't even have to play a single note in a given movement, and frequently they must sit out an entire work during a performance if their instrument isn't even needed.
