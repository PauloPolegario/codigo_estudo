Every Good Boy Deserves Fudge

The phrase "Every Good Boy Deserves Fudge" is a mnemonic to help new music students remember the names of the lines on the treble clef staff. The first line on the bottom is the note "E," and moving up from there we have G, B, D, and F at the top position. The following illustration shows the note name that each line represents:

The lines will change depending on the clef in question. Here are a few examples:

Clef Mnemonics
Clef Lines Mnemonic
Treble Clef EGBDF Every Good Boy Deserves Fudge.
Bass Clef GBDFA Good Burritos Don't Fall Apart.
Alto Clef FACEG Felix After Christmas Eats Grapes
Tenor Clef DFACE Don't Fear Alligators Cruising East.
